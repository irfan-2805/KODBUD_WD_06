# Task 6 - Responsive Image Gallery

A responsive photo gallery called **SnapGrid** built using **HTML**, **CSS Grid**, and **JavaScript** as part of my Web Development Internship at **Kodbud**.

---

## Task
**Task 6 - Create an Image Gallery**
> Make a responsive image gallery using CSS Grid or Flexbox. Show images in 3-column layout on desktop, collapse to 1-column on mobile.

---

## Features
-  3-column grid on desktop
-  2-column on tablet, 1-column on mobile
-  Image zoom on hover
-  Overlay with title & photographer name
-  Category tags on each image
-  Filter buttons (All, Nature, City, Architecture, People)
-  Working JavaScript filter functionality
-  Navbar & footer

---

## Built With
- HTML5
- CSS3 (CSS Grid)
- JavaScript (DOM filtering)
- Picsum Photos (placeholder images)
